Parallax

The parallax effect is the displacement of two images' positions, which is achieved by moving them at different rates compared to the viewer's perspective. This is a popular technique in web development and is also one of the primary techniques used to help achieve 3D effects in 3D movies (I noticed it pretty dramatically when I was watching the mountains of Mars in the opening landscape sequence of "The Martian").

The above above image is hidden behind this content block as you scroll.

The key to the parallax effect is the CSS property "background attachment". This determines whether, if a background image is specified, it should be fixed in the viewport with scrolling or whether it should follow the viewport.

By default, background attachment is set to "scroll", which means that the background will scroll along with the div element that it inhabits. But if it's set to "fixed", it will stick to the viewport itself, allowing for the cool parallax effect to occur.
