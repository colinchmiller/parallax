Parallax

The parallax effect is the displacement of two images' positions achieved by moving them at different rates in comparison to the viewer's perspective. This is a popular technique in web development and is also one of the primary techniques of helping to achieve 3D effects in 3D movies (a good example being the opening landscape shot sequence in The Martian).

The above above image is hidden behind this content block as you scroll.

The key to the parallax effect is the CSS property "background attachment". This determines whether, if a background image is specified, it should be fixed in the viewport with scrolling or whether it should follow the viewport.
